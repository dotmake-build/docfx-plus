Release notes for version 2014.1.26.0

Changes in This Release

  • Switched to the new date-based version numbering scheme.

  • Added information on the member ID prefix codes from the .NET Documentation Guidelines document.

  • Added brief information on the codecontracts elements from the Code Contracts User Manual.

  • Added async, Async, await, Await, let!, async/await, Async/Await, and async/let! as possible langword attribute values on the see element.
