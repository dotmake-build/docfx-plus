Remarks

Bicycle Power sensors fall into two main categories - StandardPowerSensorand CrankTorqueFrequencySensor.

Standard power sensors may support TorqueSensorin addition to the power only profile. The property TorqueSensor is set when a StandardCrankTorqueSensoror StandardWheelTorqueSensormessage is received.

The CrankTorqueFrequencySensorclass is somewhat unique among bicycle power sensors. It provides limited commonality and functionality compared to StandardPowerSensors.
