Requests the data page.

public Task<MessagingReturnCode> RequestDataPage<T>(T page, byte descriptor1 = 255, byte descriptor2 = 255, byte transmissionResponse = 4, CommandType commandType = CommandType.DataPage, ushort slaveSerialNumber = 65535) where T : Enum

Parameters

page T

The requested page.

descriptor1 byte

The descriptor1. The default is 0xFF.

descriptor2 byte

The descriptor2. The default is 0xFF.

transmissionResponse byte

The transmission response. The default is to send 4 messages.

commandType CommandType

Type of the command. The default is DataPage.

slaveSerialNumber ushort

The slave serial number. The default is 0xFFFF.

Returns

Task<MessagingReturnCode>

Status of the request.

Type Parameters

T

The data page enumeration of the derived ANT device class.

Exceptions

ArgumentException

Invalid data page requested.
