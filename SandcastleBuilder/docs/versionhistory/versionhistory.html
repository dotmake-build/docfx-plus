Version History

The topics in this section describe the various changes made to the Sandcastle Help File Builder over the life of the project. Select a version below to see a description of its changes.

Be aware that breaking changes may have been made since the last release you were using. Please review the release notes from all releases after it to the latest to find changes that may require you to update your project such as modifications to build component and plug-in configurations.

See the Component Migration Support topic for information on updating third-party components, plug-ins, presentation styles, and syntax generators for use with the latest version of the help file builder.
