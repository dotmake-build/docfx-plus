Version 2017.12.30.2

This is a minor update to correct the two issues noted below. See the Version 2017.12.30.0 release notes for important information on some breaking changes.

Changes in This Release

  • Modified the names and handling of the relationship files in the Open XML presentation style to work around an issue with the latest version of NuGet which adds a duplicate entry to its content types file for the .rels file type which can break third-party tools that parse the package.

  • Merged changes to fix a couple of issues with the update to the Lightweight Website Style TOC plug-in to construct TOC URLs relative to the root.
