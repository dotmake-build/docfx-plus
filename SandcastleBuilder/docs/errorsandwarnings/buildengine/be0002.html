BE0002: Shared builder content for the '[langId]' language could not be found

Warning BE0002: Shared builder content for the ' [langId] ' language could not be found. Using 'en-US, English (US)' defaults.

Cause

This warning indicates that a help file builder shared content file does not exist for the selected language.

Solution

If a language resource file does not exist, the US-English defaults are used. If you need resources to appear in the language selected in the project file, you will need to create a shared content file for the selected language. See the Localizing Language Resource Files topic for details. Once done, you can create an issue in theSHFBproject on GitHub and attach the translated help file builder resource files so that they can be included in a future release of the help file builder.

Note

Do not send any translated Sandcastle presentation style files. These are not part of the help file builder and will not be included as part of it.
