BE0011: Unable to load package reference information for '[path]'

Warning BE0011: Unable to load package reference information for '[path]'. Reason: Unable to locate targets element in project assets file '{1}'.

Cause

This error occurs if the build engine is unable to locate the targets element in a project assets file. This information is required in order to resolve package reference information.

Solution

Open an issue at the project site and attach an example that can be used to reproduce the issue.
