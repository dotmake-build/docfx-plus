Conceptual Content/Additional Content

This topic covers common questions and issues related to the conceptual content and additional content.

I want my additional content (HTML) pages to look like the selected presentation style used in theAPI topics. How can I do that?

The preferred method for adding additional content (also known as conceptual content) to a project is to use Microsoft Assistance Markup Language (MAML). To have the HTML topics match the presentation style, convert them to MAML. See the Conceptual and Additional Content topics for more information.

How do I arrange the conceptual content pages and specify which is the default topic?

The content layout file editor is used to define the content layout and the default topic.
