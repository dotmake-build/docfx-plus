Build Components

This topic covers common questions and issues related to build components.

How do I configure the custom build components with project-specific settings?

Use the Components project property category. Select the component from the list on the property page, add it to the project, and click the Configure button.

Where can I find more information on the custom build components supplied with the help file builder?

See the BuildAssembler Components help topic for more information and links to details about each component.

How can I create my own custom build components?

For information on writing your own custom build components, see the Creating a BuildAssembler Build Component topic.
