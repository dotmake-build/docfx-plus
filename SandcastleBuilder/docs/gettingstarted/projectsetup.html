Enabling and Using XML Comments

XML comments must be added to your code to document the various types and their members. In addition, you must enable XML comments file output for the project. For information on enabling XML comments in your project and decorating your code with them, see the Enabling and Using XML Comments walkthrough topic in the Sandcastle XML Comments Guide.

Once you have enabled XML comments in your project and added some comments to your code, you are ready to create your first help file builder project.
